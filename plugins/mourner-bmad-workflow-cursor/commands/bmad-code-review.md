# /bmad-code-review

Use the official BMAD `bmad-code-review` workflow bundled in this plugin.

Open `workflow/bmad-code-review/SKILL.md` (relative to this plugin root) and follow it, including any linked step files under `workflow/bmad-code-review/`, until the user's request is complete.

Do not rely on a separate "skill" activation: the workflow content is only in those files.
