# /bmad-help

Use the official BMAD `bmad-help` workflow bundled in this plugin.

Open `skills/bmad-help/SKILL.md` (relative to this plugin root) and follow it, including any linked step files under `skills/bmad-help/`, until the user's request is complete.

Do not rely on a separate "skill" activation: the workflow content is only in those files.
